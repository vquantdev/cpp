4. Determine the max element using template function

max_template(10, 20) = 20

max_template(1.99, 2.01) = 2.01

max_template('A', 'a') = a

max_template(10, 10.01) = 10 // 0.01 truncated because return type is 'T' (int in this case)

max_template(10.01, 10) = 10.01

max_template(50, 'A') = 65 // return type 'int'

max_template('A', 50) = A // return type 'char'
